# R_regression_analysis2
This regression analysis is used to predict the LBM (lean body mass) of athletes and whether or not there is a difference in LBM for males and females. The analysis uses the following explanatory variables: Sex (0: males, 1: females), Ht (height in cm), Wt (weight in kg), WCC (white cell count), Hg (hemoglobin) amnd Hc (hematocrit). This analysis was created using R programming language.
